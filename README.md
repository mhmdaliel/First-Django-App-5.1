## Installation

### Requirements

* Python 3+
* Pipenv
* Django 1.11+

### Steps

```bash
apt-get install python3 python3-pip
python3 -m venv myenv
pip install pipenv
pipenv install --three
cd djangotutorial/
python manage.py runserver
Go to : http://127.0.0.1:8000/admin/
The default login and password for the admin site is admin:admin
```
